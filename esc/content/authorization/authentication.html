SAS Authentication

SAS Client Authentication

Prior to SAS-ESC communications, the (human) ESC operator and SAS administrators must exchange the Public Key Infrastructure (PKI). In particular, at the end of this step, the ESC must posses the SAS's public key and the SAS must possess the ESC's public key.

The private keys for both entities are kept private and not shared.
