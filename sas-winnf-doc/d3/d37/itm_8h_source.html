64 // Parameter documentation:
65 // elev : an array of terrain elevation values in meters. The format is that the
66 // first array value is the total size of the elevations passed minus 1, the
67 // second is the distance between the values in meters, and then come the
68 // actual elevation values. So passing four values separated by 100m
69 // would look like: [ 3, 100, 1, 2, 3, 4]
70 // tht_m : transmitter height in meters
71 // rht_m : receiver height in meters
72 // eps_dielect : the dielectric constant (relative permittivity) of the ground.
73 // Typical values are 15 for ground, 81 for water.
74 // sgm_conductivity : conductivity of the soil/ground.
75 // Typical values are 0.005 for gruond, 0.01 for fresh water,
76 // 5 for sea water.
77 // eno_ns_surfref : refractivity of the atmosphere.
78 // Should be between 250 and 400. (Reasonable default value is 314.)
79 // frq_mhz : the emission frequency in megahertz
80 // radio_climate : the radio climate. one of the RADIO_CLIMATE_* values above
81 // pol : emission polarization. one of the POL_* values above
82 // conf : Confidence. A value between 0.01 and 0.99.
83 // This value indicates a fraction of situations that the actual loss will
84 // not exceed the model loss in similar situations.
85 // rel : Reliability. A value between 0.01 and 0.99
86 // This value indicates a fraction of the time that the actual loss will not exceed
87 // the model loss for a particular radio system.
88 
89 // ModVar : ???. One of the MODVAR_* values above
90 // deltaH : interdecile range of elevations in the path
91 // dist_km : path length distance in kilometers
92 // TSiteCriteria : site criteria for transmitter. One of the SITE_CRITERIA_* values above.
93 // RSiteCriteria : site criteria for receiver. One of teh SITE_CRITERIA_* values above.
