README

This directory contains raw and processed data files from the US Census defining outlines of census tracts used in the 2010 decennial census. Copyright on data files is by their creators.

  • tl_2010_??_tract10.zip

    Zip files containing TIGER/Line shapefiles which define census tract boundaries for the 2010 census. These are zipped files whose name references the FIPS code of the state in which that particular file's census tracts lie. The files are referenced here: https://www.census.gov/geo/maps-data/data/tiger-line.html and can be retrieved from this FTP site: ftp://ftp2.census.gov/geo/tiger/TIGER2010/TRACT/2010/
